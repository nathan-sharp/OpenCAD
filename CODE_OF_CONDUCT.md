# Code Of Conduct

## Purpose

OpenCAD is an engineering standards project. Productive review depends on precise, professional, and respectful technical discussion.

## Expected Behavior

Participants are expected to:

- focus on technical substance
- assume good faith while challenging ideas directly
- provide evidence, examples, and concrete reasoning
- keep discussions professional and relevant to the project
- respect differences in background, experience, and employer affiliation

## Unacceptable Behavior

The following behavior is not acceptable in project spaces:

- personal attacks, insults, or harassment
- discriminatory or exclusionary language
- threats, intimidation, or doxxing
- repeated off-topic disruption
- knowingly misrepresenting technical evidence or project decisions

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and other project communication channels.

## Enforcement

Maintainers may edit, hide, lock, or remove content that violates this code of conduct. Repeated or severe violations may result in temporary or permanent removal from project participation spaces.

## Reporting

Report conduct concerns privately to the maintainers listed in [MAINTAINERS.md](MAINTAINERS.md).
